# Elevate Labs Internship – Daily Task Submissions (Data Analyst)

This GitHub repository contains my **daily task submissions** for the **Elevate Labs Data Analyst Internship**.

Each task folder/ZIP includes the dataset, working files, reports, and insights as required.

---

## ✅ What This Repository Contains

For every task submission, I upload:

- **Dataset File** (CSV / Excel)
- **Work File** (Excel / Google Sheets / Python Notebook)
- **Report / Output File** (PDF / Screenshots)
- **Insights / Summary File** (TXT / DOC)

---

## ✅ Common Work Done in Tasks

Most tasks include one or more of the following:

- Data collection and dataset selection
- Data cleaning (removing blanks, duplicates, formatting corrections)
- Data preprocessing (date fixing, number conversion, trimming spaces)
- Data analysis (Pivot tables, summaries, filtering, grouping)
- Visualization (Pivot charts / graphs)
- Final insights and conclusions
- Exporting final output as PDF
- Uploading all deliverables to GitHub for submission

---

## ✅ Tools Used

- Microsoft Excel / Google Sheets  
- Pivot Tables & Pivot Charts  
- Basic Analytical Functions in Excel  
- GitHub (for submission link)

---

## ✅ Submission Process

For each task:
1. Download/select dataset
2. Clean and prepare the dataset
3. Perform analysis and generate outputs
4. Create a report (PDF) and insights file
5. Upload everything to GitHub
6. Submit the GitHub repository link in the Elevate Labs submission form
